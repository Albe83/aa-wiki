# Artificial Analysis

Artificial Analysis is an independent AI benchmarking and analysis platform. It evaluates language models, agents, media generation models, and API providers with objective metrics of intelligence, speed, and cost.

## Key facts

- Founded as an independent platform: no compensation from providers for listings or favorable results
- Primary testing server: Google Cloud us-central1-a
- Measurement standard: OpenAI Tokens (tiktoken o200k_base) for speed, native tokens for pricing
- Integrity terms: providers must not distinguish or treat AA traffic differently
- Covers: LLMs, coding agents, image/video/music generation, STT, TTS, speech-to-speech, hardware
- Fully public and transparent methodology (prompts, parameters, scoring)
- Proprietary datasets: AA-AgentTalk (STT), AA-Briefcase (knowledge work), GDPval-AA (implementation of GDPval)
