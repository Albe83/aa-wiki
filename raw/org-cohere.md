# Cohere

- **Headquarters:** Toronto, Ontario, Canada
- **Founded:** 2019
- **Founders:** Aidan Gomez, Nick Frosst, Ivan Zhang
- **Key models:** Command series (Command, Command R, Command R+)
- **Valuation:** ~$7 billion (2025)
- **Employees:** ~450+
- **Focus:** Enterprise AI for regulated industries (finance, healthcare, government)
- **Notable:** Aidan Gomez is a co-author of the "Attention Is All You Need" paper. Cohere emphasizes enterprise security, data privacy, and retrieval-augmented generation (RAG) deployments.
- **Source:** https://en.wikipedia.org/wiki/Cohere
